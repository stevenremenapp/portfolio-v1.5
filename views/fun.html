THIS SITE

TOOLS

  • HTML
  • CSS
  • JS
  • p5.js
  • Node + Express Server

This site grew from needing to update my portfolio, being interested in learning some fun JavaScript libraries, and having my niece stay with us for a while around that same time. She was showing me some of the games and sites she liked, and I thought we might be able to use p5.js to make something similar. After about a week of working together we had a prototype to present at a Shift_Up demo day!

This site started there, and then I liked the canvas enough to see if I could extend it to an entire site. After working pretty exclusively with frontend JavaScript frameworks and Sass for a while I wanted to limit my tools on the frontend. I also took this opportunity to learn more about CSS Grid and practice my design skills.

Direct inspiration for various parts of this site came from: Ali Spittel, Yaeji, The Coding Train, Mackenzie Child (Flowmingo), Marguerite Roth, Brittany Chiang, Monica Dinculescu, Jens Nielsen, and Lynn Fisher.

JEAN-RALPHIO QUOTE MACHINE

TOOLS

  • HTML
  • CSS
  • JS
  • Photoshop

This is a quote machine created by suggestion of freeCodeCamp, but in making it I realized that the internet needed a quote machine from Jean-Ralphio of Parks and Rec.

This was my first JavaScript project and taught me how to create random numbers and strip HTML from a string. I also learned a lot of useful CSS, design, and Photoshop tricks while finishing this project. Although this is not my most technically demanding project, it is by far the project that returns the most enthusiastic response.

There was a time when I was really into CSS illustrations while also not being particularly good at them. This is my best effort and the running animation still makes me happy. It's exactly as I remember him from my childhood!

CLUEDO: AN UNAUTHORIZED REWORKING OF CLUE

This game was created years ago with a couple of close friends as an attempt to insert political economy into the Clue board game.

We started with the original Clue and added currency, banking, and the option to purchase "abilities," such as wire tapping, transporting, looking at solution cards, and the Bag-O-Death.

Click here for full instructions and to read more. If you try it out, let me know how it goes!
